Code

The Pygments package can very easily highlight Python code using different patterns:

import numpy as np
np.ones((10, 10))
res = a + b

This is true for a lot of languages, including C++:

void test(){
    int i = 0;
    for(int i=0; i<10; i++){
        sleep(1);
    }
    std::cout << "Hello, World!" << std::endl;
}
